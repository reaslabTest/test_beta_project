# 1车多客户同时装卸货车辆路径问题 (1-M-1-PDP)

## 问题描述

某物流公司负责为一个区域内的多个客户进行货物配送和回收。公司的物流中心（Depot）每天需要向这些客户配送不同数量的商品（卸货需求），并同时从客户处回收不同数量的物资（装货需求）。

公司拥有一支由多辆货车组成的车队，每辆车的载重能力都是相同的。现在的任务是为这些货车规划出最优的行驶路线，目标是最小化所有车辆行驶的总距离。在规划路线时，必须满足以下所有条件：

1.  **服务范围**：所有需要服务的客户都必须被访问，且只能被一辆车访问一次。
2.  **车辆路径**：每辆车都必须从物流中心出发，完成其配送和回收任务后，最终返回到物流中心。
3.  **容量限制**：在行驶过程中的任何时刻，车上装载的货物（包括待配送的商品和已回收的物资）总重量不能超过车辆的载重上限。
4.  **需求满足**：
  *   必须满足每个客户的所有卸货需求。
  *   必须满足每个客户的所有装货需求。

这是一个典型的带同时装卸货（Simultaneous Pickup and Delivery）的车辆路径问题（VRPSPD）。

### 数据说明

*   **节点信息**：包括一个物流中心（Depot）和多个客户节点。每个节点都有其地理坐标。
*   **需求信息**：
  *   每个客户节点有明确的卸货（Delivery）需求量。
  *   每个客户节点有明确的装货（Pickup）需求量。
*   **车辆信息**：车队包含多辆车，所有车辆的载重能力（Capacity）相同。
*   **距离**：任意两个节点之间的行驶距离根据其坐标计算得出。
